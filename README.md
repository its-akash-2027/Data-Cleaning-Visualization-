# Data Cleaning & Visualization Project

## Project Overview
This project focuses on cleaning, preprocessing, and visualizing the Titanic dataset using Python. The goal is to improve data quality and extract meaningful insights through data visualization.

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn

## Project Tasks
- Loaded the Titanic dataset
- Removed duplicate records
- Handled missing values
- Removed outliers using the IQR method
- Encoded categorical variables
- Created data visualizations
- Generated key insights
- Saved the cleaned dataset

## Visualizations
- Histogram
- Box Plot
- Bar Chart
- Scatter Plot
- Correlation Heatmap
- Pie Chart

## Key Insights
- Cleaned and preprocessed the dataset.
- Removed missing values and duplicate records.
- Visualized relationships between features.
- Identified trends in passenger survival.

## Outcome
This project demonstrates practical skills in data preprocessing, exploratory data analysis (EDA), and data visualization using Python.
